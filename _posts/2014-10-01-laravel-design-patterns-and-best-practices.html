Autor</strong>: Arda Kılıçdağı, H. İbrahim YILMAZ
Editorial</strong>: Packt Publishing
Páginas</strong>: 106
Fecha de publicación</strong>: Julio 2014
ISBN</strong>: 9781783287987
Dirigido a</strong>: Desarrolladores que trabajan con Laravel que desean aumentar la eficiencia de sus aplicaciones
Nivel requerido</strong>: Intermedio
Rating</strong>: 9/10
Pros</strong>: Ejemplos prácticos muy buenos para implementar patrones de diseño en aplicaciones con Laravel
Cons</strong>: Algunos capítulos mencionan explicaciones que se pueden encontrar en la documentación

Laravel</a> se está convirtiendo en uno de los frameworks PHP</a> mas utilizados hoy en día por miles de desarrolladores. Yo lo recomiendo totalmente para quien quiera desarrollar aplicaciones web de una manera elegante, sencilla y expresiva, siguiendo con principios SOLID</a>.

El libro Laravel Design Patterns and Best Practices</a> aborda de una forma práctica y sencilla los principios de utilización de patrones de diseño en aplicaciones web utilizando de base a Laravel como framework de desarrollo. Para el que no este muy familiarizado con los patrones de diseño, el autor hace una breve explicación de cada uno de ellos, y menciona casos prácticos en los que estos se pueden utilizar en nuestras aplicaciones.

Se muestran ejemplos de clases ya presentes en el framework que hacen uso de algún patron de diseño, con esto nos permite entender de manera más visual a través del código ya implementado en el framework.

Laravel utiliza el patrón de arquitectura de software MVC</a>, por lo que el autor hace una extensiva explicación de como se implementa este en el framework, y como funcionan los controladores, modelos y vistas. Además nos da varias recomendaciones de cuales pueden ser las mejores prácticas que podemos utilizar en nuestro código, para ayudarnos a que nuestras aplicaciones puedan crecer adecuadamente, tener todo bajo control y hacer las cosas como deben de ser.
